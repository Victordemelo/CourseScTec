Crie uma página HTML básica com um título, descrição da atividade e espaço para inserção de dados pelo usuário.

Adicione dois campos de entrada (input): um para o preço original do produto e outro para a porcentagem de desconto.

Inclua um botão que, ao ser clicado, execute uma função JavaScript para calcular o preço final após o desconto.

A função deve usar operadores para calcular o valor do desconto e o preço final.

Use estruturas condicionais if-else para verificar se os valores inseridos são válidos (por exemplo, preços ou porcentagens negativas não são permitidas).

Exiba o resultado usando console.log e alert para mostrar o preço original, o valor do desconto e o preço final.
